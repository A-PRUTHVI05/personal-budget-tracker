Personal Budget Tracker is a lightweight command-line tool built with Python to help you manage your finances.
It allows you to:

Record income and expenses with date, category, and description

View and filter transactions over custom date ranges

Automatically generate financial summaries

Visualize trends with matplotlib graphs

Store data locally in a CSV file for easy backup and portability

This project is ideal for those who prefer a simple, offline solution for tracking finances using familiar tools like Python and CSV.
